# Simple Chatbot UI

A lightweight chatbot interface built using HTML, CSS, and JavaScript inside a single `index.html` file. This project demonstrates how a simple interactive chat interface can be created without using frameworks or external libraries.

---

## Features

- Simple chat-style interface
- User message input
- Dynamic message display
- Lightweight implementation in a single file
- Beginner-friendly project

---

## Tech Stack

- HTML
- CSS
- JavaScript

---

## Project Structure

```
react-chatbot
└── index.html
```

The entire chatbot UI and logic are implemented inside `index.html`.

---

## How to Run

1. **Clone the repository**

```bash
git clone https://github.com/BannyVardhanReddy/react-chatbot.git
```

2. **Navigate to the project folder**

```bash
cd react-chatbot
```

3. **Open `index.html` in your browser.**

No installation or dependencies are required.
